GIC-4 (Gender Intensity Code 4)

An attempt to quantify gender expression by creating 100 million genders, while maintaining human readability.

Take Quiz

GIC-4 (Gender Intensity Code - 4) is a compact four-pair numeric code that quantizes a person’s gender across:
  • Presentation
  • Chest
  • Genitals
  • Skin/Hair
Each pair uses:
  • Tens digit: direction (0 = fully female → 9 = fully male)
  • Ones digit: intensity (0 = minimal → 9 = high)
Use "-" to omit intensity when not applicable.

A cis woman might have a GIC-4 code of 27,08,08,12 What does it mean?
  • 26: this person is very feminine overall.
  • 08: this person has very feminine-looking breasts, and quite big.
  • 08: this person has very feminine-looking genitalia, and quite developed.
  • 12: this person has very soft skin, and not hairy.
An agender person might have a GIC-4 code of -8,62,-5,00
  • -2: this person doesn't believe in the binary sex spectrum; however, they do believe in the concept of gender; it's just that we have no experience.
  • 62: this person has an ambiguous flat chest.
  • -5: this person's private parts are neither found on male nor female; it's a different expression entirely.
  • 00: this person has very soft hairless skin.
A gender fluid person might have a GIC-4 code of (27,08,08,12)-(87,86,77,00) They can express their gender with multiple GIC-4 codes, switching between them as they feel. If your gender is a collapsible wave function, you can write it in bra-ket notation. Example: |Ψ⟩ = c₁|(27,08,08,12)⟩ + c₂|(87,86,77,00)⟩ This will represent gender superposition.

There are no numeric representations for the wide gender spectrum and complicated nonbinary definitions. This tool can help people (not just trans or nonbinary):
  • Provides a more granular understanding of gender identity and expression, moving beyond the binary framework.
  • Acknowledges the complexity of gender, including the nonbinary spectrum, agender, demigenders, and more!
  • Allows individuals to exchange their gender easily.
And most importantly, it expresses that gender is a human-made label. You don't have to fit into any predefined boxes. You can pick and choose your own.

The color is generated from the GIC-4 code, mapping each pair to the digits to HSLA or RGBA values after normalization. It is just a visual representation of the code, and has no inherent meaning. You can switch between HSLA and RGBA views.
