GIC-4 (Gender Intensity Code 4)

An attempt to quantify gender expression by creating 100 million genders, while maintaining human readability.

Take Quiz

GIC-4 is a compact four-pair numeric code that quantizes a person’s gender goal across:
  • Presentation
  • Chest
  • Genitals
  • Skin/Hair
Each pair uses:
  • Tens digit: direction (0 = fully female → 9 = fully male)
  • Ones digit: intensity (0 = minimal → 9 = high)
Use “-” to omit intensity when not applicable.

GIC-4 breaks down gender expression into four key areas: overall, chest, genitals, and skin/hair. Each area is represented by a pair of digits, allowing for a nuanced representation of an individual's gender goals.

  • Provides a more granular understanding of gender identity and expression, moving beyond the binary framework.
  • Acknowledges the complexity of gender.
  • Allows individuals to exchange their gender goal easily.
